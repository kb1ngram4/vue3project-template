# vue3project-template

## 简介
```
基于vue3.0脚手架
```
### axios配置（二次封装）
```
yarn add axios
```
## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
